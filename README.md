# Korpus-Bahasa-Kudus
Dataset ini merupakan korpus paralel primer yang disusun secara manual untuk mendukung penelitian penerjemahan otomatis dari Bahasa Indonesia ke Bahasa Daerah Kudus. Dataset ini ditujukan untuk melestarikan dialek Kudus melalui teknologi Natural Language Processing (NLP) dan sangat cocok digunakan untuk eksperimen model dengan sumber daya rendah.
